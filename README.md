# Desafio de Projeto 2 - Criando um Relatório de Vendas e Lucros com Power BI

## 📌 Descrição do Projeto
Este projeto faz parte da **Formação Power BI Analyst** da DIO.me. O objetivo principal é a criação de um relatório dinâmico e visualmente estruturado no Power BI, focado na análise de vendas e lucros. O desafio consistiu em transformar uma base de dados bruta em um dashboard funcional, permitindo a exploração de métricas de desempenho por produto, segmento e período temporal.

## 📊 Estrutura do Relatório
O dashboard foi dividido em páginas para facilitar o consumo de informações pelo cliente final:

### 1. Página Principal (Home/Overview)
Focada em métricas de alto nível para uma visão rápida da saúde financeira:
- **Total de Vendas:** R$ 118,73 Mi.
- **Unidades Vendidas:** 1,13 Mi.
- **Custos (COGS):** R$ 101,83 Mi.
- **Descontos Totais:** R$ 9,21 Mi.
- **Vendas por Período:** Gráfico de linha demonstrando a sazonalidade ao longo dos meses.
- **Vendas por Segmento:** Distribuição entre Government, Small Business, Enterprise, Midmarket e Channel Partners.

### 2. Página de Detalhes
Uma visão granular dos dados conforme solicitado no desafio:
- **Top 3 Produtos:** Identificação dos itens de maior performance (Paseo, VTT e Velo).
- **Vendas por País:** Mapa e gráficos de barras detalhando o desempenho geográfico (USA, França, Canadá, Alemanha e México).
- **Análise de Lucro (Profit):** Comparativo entre as vendas e a margem de lucro por categoria.

### 3. Análise de Dados (Visualizações Avançadas)
- **Histograma:** Distribuição das unidades vendidas.
- **Gráfico de Dispersão:** Relação entre Unidades Vendidas e Vendas por mês para identificar correlações.
- **Segmentação (Slicers):** Filtros dinâmicos por data e categoria para exploração customizada.

## 🛠️ Tecnologias e Ferramentas
- **Power BI Desktop:** Criação de visuais e modelagem.
- **DAX (Data Analysis Expressions):** Criação de medidas calculadas personalizadas.
- **Power Query:** Limpeza e tratamento dos dados.

## 📈 Pontos Considerados no Desenvolvimento
- **Experiência do Usuário (UX):** Disposição lógica dos visuais para que a leitura dos dados ocorra de forma fluida.
- **Métricas:** Criação de medidas essenciais para análise de profit e margem.
- **Design:** Uso de paleta de cores consistente e navegação entre páginas através de botões.
